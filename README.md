## 🚀 About The Project

This project is a **full-featured Task Manager application** developed using the **MERN (MongoDB, Express.js, React, Node.js) stack**.
It demonstrates a **real-world full-stack web app** with robust backend APIs and a responsive, modern frontend.

👉 The main goal was to create a **practical project for job portfolio & freelancing**, while showcasing clean architecture, authentication, and collaboration features.

---

## ✨ Key Features

* 🔐 **User Authentication** – Secure registration & login with JWT
* 👨‍💼 **Role-Based Access** – Admin can manage teams, users can manage tasks
* ✅ **CRUD Functionality** – Create, Read, Update, Delete tasks
* 📅 **Task Management** – Add tasks, set priority & deadlines
* 👥 **Team Collaboration** – Assign tasks to team members
* 📱 **Responsive UI** – Works across desktop, tablet, and mobile
* ⚡ **RESTful API** – Clean and structured Express.js APIs
* 🎨 **Modern Design** – React + Tailwind CSS for smooth UX

## ⚙️ Installation & Setup

Clone the repo:

```bash
git clone https://github.com/mirza-umair-dev/Mern-Task-manager.git
cd Mern-Task-manager
```

Install dependencies:

```bash
# Install backend dependencies
cd backend && npm install

# Install frontend dependencies
cd frontend && npm install
```

Run the app locally:

```bash
# Start backend
cd backend && npm run dev

# Start frontend
cd frontend && npm start
```

App runs on: **[http://localhost:3000](http://localhost:3000)**
